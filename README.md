# Rediseño Web - Escola Lluís Millet

## 📋 Descripción

Rediseño completo y desarrollo de la nueva web para la **Escola Lluís Millet**, un centro educativo concertado en Barcelona que imparte Infantil, Primaria y Secundaria.

Este proyecto responde a una petición real de la dirección del centro, reemplazando una plataforma obsoleta (blocs.xtec.cat) por una solución moderna, accesible (WCAG 2.1), responsiva y conforme al RGPD.

**Proyecto Final de Ciclo Superior en Desarrollo de Aplicaciones Multiplataforma (DAM)**

---

## 🛠 Tecnologías Utilizadas

### Frontend
- **Next.js 16** - Framework React con App Router
- **React 19** - Librería de UI
- **TypeScript** - Tipado estático
- **Tailwind CSS + Bootstrap** - Estilos y diseño responsivo
- **Vercel** - Hosting y despliegue

### Backend
- **Strapi 5.31.2** - CMS Headless
- **Node.js 18** - Runtime
- **PostgreSQL** - Base de datos
- **Render** - Hosting y despliegue

---
